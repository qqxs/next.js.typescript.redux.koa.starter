![build](https://github.com/qqxs/next.js.typescript.redux.koa.starter/workflows/Next.js%2Typescript%2Redux%2Koa%2starter/badge.svg)

## Use

```sh
npm install

# run development
npm run dev

# build production
npm run build

# build production and export static source
npm run export

# run production
npm run start:prod
```
